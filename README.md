# 📝 Interactive Task Manager

An interactive and responsive web application built using **HTML, CSS, and JavaScript** that allows users to manage their daily tasks efficiently. The application provides real-time updates, dynamic content rendering, and a smooth user experience without requiring page reloads.

---

## 🚀 Features

- ➕ Add new tasks instantly
- ✅ Mark tasks as completed
- 🗑️ Delete tasks dynamically
- 📊 Live task counter
- 📱 Fully responsive design
- ⚡ Real-time UI updates
- ✔️ Form validation
- 🎨 Clean and modern user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🎯 How It Works

1. Enter a task in the input field.
2. Click the **Add Task** button.
3. The task is displayed instantly in the task list.
4. Click on a task to mark it as completed.
5. Click the **Delete** button to remove a task.
6. The task counter updates automatically.
